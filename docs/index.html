MainsailCNC

A modern, responsive CNC controller interface for Klipper-based machines.

Purpose

Mainsail CNC is a fork of Mainsail — the popular web interface for 3D printers running Klipper — adapted and extended for CNC machine control.

It provides a real-time, browser-based control panel for Klipper-based CNC mills, routers, plasma cutters, and lathes. Built with Vue 3 and Vuetify 3, it runs on any device with a web browser — phone, tablet, or desktop.

Key Features

CNC Dashboard

Machine status, DRO, jog controls, WCS offsets, spindle & coolant control at a glance.

Temperature Management

Monitor and control spindles, enclosures, and additional sensors with real-time charts.

G-Code File Management

Upload, organize, and send G-code files directly to your machine. Thumbnail previews included.

MDI Console

Send raw G-code commands interactively with autocomplete and command history.

Webcam Support

Multiple webcam streams with support for MJEG-Streamer, camera-streamer, and more.

History & Analytics

Track job history, print time averages, and machine usage statistics.

Machine Config

Edit Klipper config files, manage Moonraker settings, and update firmware from the browser.

Farm Mode

Monitor and control multiple machines from a single dashboard.

Ansible Deployment

Idempotent install & upgrade via Ansible playbooks. Pre-built releases for low-RAM devices.

Theming

Customizable UI with built-in themes and full branding support.
